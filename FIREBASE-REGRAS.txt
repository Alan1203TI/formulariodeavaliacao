REGRAS DO FIRESTORE PARA ESTE SISTEMA FLL

Cole SOMENTE o bloco abaixo no Firebase Console:
Firebase > Firestore Database > Regras > Publicar

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {

    match /usuarios/{userId} {
      allow read: if true;
      allow write: if true;
    }

    match /avaliacoes/{doc} {
      allow read: if true;
      allow write: if true;
    }

    match /equipes/{doc} {
      allow read: if true;
      allow write: if true;
    }

    match /salas/{doc} {
      allow read: if true;
      allow write: if true;
    }
  }
}

IMPORTANTE:
- Esta regra não possui data de expiração.
- As coleções usadas pelo código são: usuarios, avaliacoes, equipes e salas.
- Não publique este arquivo no lugar do index.html no GitHub Pages.
