REGRAS DO FIRESTORE PARA ESTE SISTEMA FLL

Cole SOMENTE o bloco abaixo no Firebase Console:
Firebase > Firestore Database > Regras > Publicar

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /avaliacoes_fll/{doc} {
      allow read, write: if true;
    }
    match /usuarios_fll/{doc} {
      allow read, write: if true;
    }
  }
}

IMPORTANTE:
- Não coloque este texto dentro do GitHub como página principal.
- O arquivo principal do site precisa ser o index.html.
- Não use regra com request.time < timestamp.date(...), pois ela expira e bloqueia leitura/salvamento.
